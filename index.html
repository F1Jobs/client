FAQs


1. What is the purpose of this site?

If you are looking for a job at one of the Formula 1 teams, browsing their individual pages for job listings gets tedious. So I made this site to display it all in one place, and apply for various positions easily. I've seen similar sites exist, but either they are not updated frequently or diverge too far away from the current F1 teams.


2. How frequently does it get updated?

Any update to job listings at original sites gets updated here within 30 minutes. Job listings don't change dramatically in a short period of time, so hitting the original sites for instantaneous updates would only make this site slower.


3. Fast, you say. How so?

  • This is a static website. It uses the Fetch API to get stored data from a web server, as opposed to gathering it at load time.
  • It is hosted on Github Pages, which rather than serving the content directly, serves it using a global CDN.
  • It is light on resources it transfers over the network. There are no scripts for tracking or analytics.


4. Can I see the source?

Sure, the website is open-source under the GNU General Public License v3.0. The source code is hosted on GitHub.

6. I found an issue with the site or something stopped working. What can I do about that?

If something breaks here (most likely due to change in the structure of careers page of one or more F1 teams), please let me know about the problem using one of the contact methods mentioned above. You can also file an issue or open a pull request. I will look into it try to fix it as soon as possible.
